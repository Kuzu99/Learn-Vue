# Learn Vue2 and Vue3

- Vue2とVue3
- Options APIとComposition API
- など学ぶためにコードをまとめるリポジトリ

# 本環境を作った手順

1. VueCLIのInstall
```
$ npm install @vue/cli
```

2. Projectの作成
```
$ vue create vue2-introduction
```
- Vue2, Vue3, Customと出るがVue2を選択する

3. localhostにサーバーを立てる
```
$ cd vue2-introduction
$ yarn serve
```
